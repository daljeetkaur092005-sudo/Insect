This project is a simple and educational webpage layout built with HTML and CSS to present information about insects in a clean and structured format. The design focuses on readability, visual appeal, and easy navigation.

✨ Features
Header Section: Contains the title of the webpage and a navigation menu for quick access to different insect categories.

Main Content Area: Displays insect cards with images, names, and short descriptions. Each card highlights unique characteristics of insects such as beetles, butterflies, and ants.

Sidebar: Provides quick facts or additional resources about insects.
